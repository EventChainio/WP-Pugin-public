== Description ==

Sell your event tickets on your WordPress website.

== Installation ==

1. Upload `eventchain.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Edit plugin Settings and enter your EventChain email address
1. Locate the event you want to display and copy its shortcode to the clipboard ('[eventchain id=xx]')
1. Open a page/post for editing
1. Somewhere in the content paste the shortcode
